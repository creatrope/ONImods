AutoSweeperTempFilter - Oxygen Not Included Mod
===============================================

https://github.com/creatrope/ONImods/releases/tag/v0.1

Installation Instructions (Manual - Local Folder):

1. Locate your Oxygen Not Included mods directory:

   Windows:
   C:\Users\<YourUsername>\Documents\Klei\OxygenNotIncluded\mods\Local

   macOS:
   ~/Library/Application Support/unity.Klei.Oxygen Not Included/mods/Local

2. Unzip the contents of this zip file into a subfolder inside the `Local` directory.

   For example:
   C:\Users\<YourUsername>\Documents\Klei\OxygenNotIncluded\mods\Local\AutoSweeperTempFilter

   The folder should contain:
   - mod.yaml
   - mod_info.yaml
   - AutoSweeperTempFilter.dll
   - anim\ (directory with animation assets)

3. Launch Oxygen Not Included.

4. On the main menu, click "Mods".

5. Find “AutoSweeperTempFilter” in the list and enable it.

6. Restart the game when prompted.

-----------------------------------------------------

Enjoy!
